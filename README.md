# paddle-copy

A Paddle server ocr demo that copies text to the clipboard.

OcrData is generated from the PaddleX OCR model.

You can select an area to copy the text to the clipboard.

You can deploy Paddle Server from https://boommanpro.cn/post/paddle-server-ocr-client-copy .
